## 1. 概要

**ラベル一発作成**は、ExcelやCSVのデータから宛名ラベル・名札・卓上プレートのPDFを素早く作成するアプリです。

### できること

- 宛名ラベル（氏名あり・氏名なし・事業所名のみ）
- 名札（A-ONE 51002 対応）
- 卓上事業所名プレート（A4横長4分割）
- Excelからコピー＆貼り付けでデータ取込
- CSVファイルからデータ取込
- 郵便番号の自動補完（住所から検索）
- フリガナの自動補完

---

## 2. 基本的な使い方

### 手順の流れ

```
① アプリを起動
② 「＋ 新規作成」をクリック
③ データ名を入力
④ モードを選択
⑤ データを取り込む（貼り付け または CSV）
⑥ 必要に応じて編集・補完
⑦ 「PDF を出力する」をクリック
```

### 起動方法

以下のいずれかの方法でアプリを起動します。

- **デスクトップ**の「ラベル一発作成」ショートカットをダブルクリック（インストール時にショートカット作成を選んだ場合）
- **スタートメニュー** →「ラベル一発作成」をクリック

---

## 3. モードの選び方

新規作成ダイアログ上部のラジオボタンでモードを選択します。

| モード | 出力内容 | 用途 |
|--------|---------|------|
| **宛名ラベル（氏名あり）** | 郵便番号・住所・事業所名・役職・氏名　様 | 個人宛の封書 |
| **宛名ラベル（氏名なし）** | 郵便番号・住所・事業所名　御中 | 法人宛の封書 |
| **宛名ラベル（事業所名のみ）** | 事業所名　御中 | 社内便・簡易ラベル |
| **名札** | 事業所名・役職・氏名（大きめ） | 会議・研修の名札 |
| **卓上事業所名プレート** | 事業所名を均等割付（上下反転セット） | 卓上プレート |

### モード別の推奨用紙

| モード | 推奨用紙 |
|--------|---------|
| 宛名ラベル（氏名あり・なし・のみ） | A-ONE 28185 または A-ONE 28187 |
| 名札 | A-ONE 51002 |
| 卓上事業所名プレート | A4 横長4分割 |

> モードを選ぶと用紙の選択が自動的に切り替わります。

## 4. データの取り込み方

### 4-1. Excelから貼り付ける（推奨）

1. Excelでデータ範囲を選択してコピー（Ctrl+C）
2. アプリの「**貼り付けから取込**」ボタンをクリック
3. **列の対応を設定**ダイアログが開くので、各フィールドに対応する列を選択
4. 「取込む」をクリック

**推奨する列の順番（ヘッダーあり）:**

```
事業所名 | 郵便番号 | 住所 | 所属・役職名 | 氏名
```

ヘッダー行がない場合は列数で自動判定します：

| 列数 | 自動割り当て |
|------|------------|
| 2列 | 事業所名、氏名 |
| 3列 | 事業所名、肩書、氏名 |
| 4列 | 事業所名、住所、肩書、氏名 |
| 5列 | 事業所名、郵便番号、住所、肩書、氏名 |
| 6列 | 事業所名、郵便番号、住所1、住所2、肩書、氏名 |

### 4-2. CSVから取り込む

1. 「**CSV から取込**」ボタンをクリック
2. CSVファイルを選択
3. **列の対応を設定**ダイアログで列を割り当て
4. 「取込む」をクリック

> 文字コードは UTF-8 / Shift-JIS どちらも自動判定します。

### 4-3. 手動で行を追加する

「**＋ 行を追加**」ボタンで空行を追加し、各セルを直接入力できます。

- **事業所名**・**所属・役職名** のセルは改行（Alt+Enter）に対応しています
- 入力確定は Enter または Tab

### 4-4. 既存データに追記する

取り込み時に既存データがある場合は確認ダイアログが表示されます：

- **上書き**: 現在のデータをすべて削除して取り込む
- **追記**: 現在のデータの後ろに追加する

## 5. 補完機能

### 5-1. 郵便番号を自動入力

住所が入力されていて郵便番号が空の行を対象に、インターネット経由で郵便番号を検索・補完します。

1. 「**郵便番号を自動入力**」ボタンをクリック
2. 自動補完が実行される（インターネット接続が必要）
3. 完了後、補完件数と未特定件数が表示される

> 郵便番号が特定できなかった行は手動で入力してください。

### 5-2. フリガナを自動入力

事業所名が入力されていてフリガナが空の行を対象に、カタカナのフリガナを自動生成します。

1. 「**フリガナを自動入力**」ボタンをクリック
2. 完了後、補完件数が表示される

> 株式会社・有限会社などの法人格は除いて変換されます。

## 6. PDF出力

### 出力手順

1. データを確認する（チェックボックスがオンの行だけ出力されます）
2. 画面下部で**用紙**と**フォント**を選択する
3. 「**PDF を出力する**」ボタンをクリック
4. 保存先とファイル名を指定する
5. 出力完了後、PDFを開くか確認ダイアログが表示される

### 出力対象の選択

- テーブル左端のチェックボックスで行ごとに出力のオン/オフを切り替えられます
- ヘッダー行のチェックボックスで全行をまとめてオン/オフできます
- **Shift+クリック**で範囲選択ができます

### 卓上プレートモードの特記事項

A4用紙横長4分割で出力します。上下を逆向きにした2枚が1セットで印刷されるため、半分に折って両面から読めるプレートになります。

## 7. ラベル用紙・フォントの選択

### 対応用紙一覧

| 用紙キー | 用紙名 | サイズ | 配置 |
|---------|-------|-------|------|
| A-ONE 28185 | A-ONE 28185 | 70×42.3mm | 3列×6行（18面） |
| A-ONE 28187 | A-ONE 28187 | 84×42mm | 2列×6行（12面） |
| A-ONE 51002 | A-ONE 51002 | 91×55mm | 2列×5行（10面・名札） |
| A4 横長4分割 | A4横長4分割 | 200×74.25mm | 1列×4行 |

### 対応フォント

| フォント名 | 特徴 |
|-----------|------|
| MSPゴシック | 標準ゴシック体（デフォルト） |
| MSP明朝 | 明朝体 |
| メイリオ | 視認性の高いモダンフォント |

## 8. データの管理

### 一覧画面の操作

アプリ起動時に表示される一覧画面で、これまでに保存したラベルデータを管理できます。

| 操作 | 方法 |
|------|------|
| 編集（再出力） | 行をダブルクリック |
| 削除 | 「削除」ボタンをクリック |
| 複数削除 | チェックボックスで選択 → 「チェックした項目を削除」 |
| 並び替え | 列ヘッダーをクリック（クリックのたびに昇順/降順が切り替わります） |

### データの保存場所

データベースは以下に保存されます：

- **インストール済み版**: `%APPDATA%\LabelIppatsusaku\label_ippatsusaku.db`
- **開発版（bat起動）**: アプリのフォルダ直下

## 9. よくある質問

**Q. 印刷位置がずれる**  
A. お使いのプリンターのプロパティで「実際のサイズ」「拡大縮小なし」に設定してください。PDFビューアーの印刷設定でも「ページに合わせる」ではなく「実際のサイズ」を選択してください。

**Q. 郵便番号の自動入力が動かない**  
A. インターネット接続が必要です。社内ネットワークのプロキシ設定が原因の場合は、IT担当者にご相談ください。

**Q. フリガナが正しく変換されない**  
A. 固有名詞や特殊な読みは自動変換が難しいことがあります。変換後のフリガナを手動で修正してください。

**Q. 事業所名が長くて1行に収まらない**  
A. 事業所名セルを編集して、折り返したい箇所で **Alt+Enter** を入力すると任意の位置で改行できます。

**Q. 出力したいデータだけを選びたい**  
A. 各行左端のチェックボックスを使って出力対象を選択してください。チェックがオフの行はPDFに出力されませんが、データとして保存はされます。

**Q. アプリが自動更新されない**  
A. アプリ起動時に上部にアップデート通知が表示されます。通知が表示された場合は案内に従ってインストーラーを実行してください。
